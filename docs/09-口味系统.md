# 09 - 口味系统

在《Dungeon Tavern》中，酿酒原材料所蕴含的风味**完全与战斗、探索状态无关**。它们**仅仅作为酒水的口味倾向指标**，用来在夜间经营时判定人类与不同魔物顾客的口味契合度，从而决定其消费满意度。

---

## 1. 16 种核心口味定义

所有的口味倾向纯粹用以描述酒水的感官风味，分为“基础口味”、“体感口味”与“地底特异口味”三大类：

| 序号 | 口味名称 | 英文标识 | 描述 (纯感官风味表现) |
| :--- | :--- | :--- | :--- |
| **1** | **甜美** | Sweetness | 莓果、花蜜、野生糖分带来的愉悦甘甜。 |
| **2** | **酸爽** | Acidity | 野生果酸、柠檬或植物发酵产生的微酸气泡感。 |
| **3** | **苦涩** | Bitterness | 苔藓、树皮、精制啤酒花带来的清苦或涩口味道。 |
| **4** | **辣口** | Spiciness | 辣椒、辣根等强刺激性原料带来的发热痛觉感官。 |
| **5** | **咸度** | Salty | 地底岩盐、盐结晶带来的咸香口感。 |
| **6** | **香醇** | Aromatic | 植物花粉、芳香作物带来的复合香气与回甘。 |
| **7** | **麦香** | Maltiness | 粮食、大麦、块茎重发酵产生的粮食基底熟香。 |
| **8** | **果香** | Fruity | 地底新鲜浆果带来的自然清甜果味。 |
| **9** | **温暖** | Warmth | 流经喉咙时，高浓度或温热基底带来的融融体感。 |
| **10** | **寒凉** | Coolness | 冰霜结晶、发光植物带来的薄荷般彻骨冰凉体感。 |
| **11** | **浓郁** | Richness | 酒体黏稠、挂杯、发酵极其充分的醇厚口感。 |
| **12** | **清澈** | Crispness | 口感干净利落，无杂质，极易入口。 |
| **13** | **恶臭** | Stench | 大脑排斥的刺鼻腐臭味，如地底怪物的下水料味道。 |
| **14** | **腐败** | Rot | 熟透至发霉霉变的菌菇、坏掉的水果产生特有酸坏味。 |
| **15** | **死寂** | Decayed | 冰冷死灰、凋零遗迹带来的虚无腐朽气味。 |
| **16** | **剧毒** | Poisonous | 危险毒素作物带来的微麻与刺激性辣痛感。 |

## 2. 种族口味期望阈值矩阵

本作彻底废除复杂的“加权乘法计算公式”。判定顾客是否满意，采用**“期望阈值门槛判定法”**——只要酒水中顾客**喜爱的口味强度均大于或等于（$\ge$）其期望阈值**，并**讨厌的口味强度全部为 0**，即可直接判定为 **【满意/喝爽】**（顾客掏空钱包 $M$，好感度暴增）。

```
【满意度极简判定逻辑】
 满足判定 = (酒水喜爱口味 A >= 期望 A) && (酒水喜爱口味 B >= 期望 B) && ... && (酒水讨厌口味 == 0)
```

*(注：在运行时动态生成怪物顾客时，其喜好的口味期望值会有 $\pm 1$ 的个体点状微调，以下为种族基础母本偏好阈值表)*

### 2.1 各种族基础偏好阈值表

*   **人类冒险者 (明码标价定价制 - 无好感系统)**：
    *   **喜爱风味与期望阈值**：`[麦香: >= 2]` && `[甜美: >= 1]`
    *   **绝对排斥风味 (一票否决)**：`[恶臭]`、`[腐败]`、`[死寂]`、`[剧毒]` (必须全为 0)
    *   *说明*：只要不含有魔物恶癖风味，同时麦香与甜美达标，人类即会满意并支付菜单全额金币。
*   **哥布林 (魔物倾囊制 - 偏好赠予材料)**：
    *   **喜爱风味与期望阈值**：`[腐败: >= 2]` && `[甜美: >= 1]`
    *   **讨厌风味 (一票否决)**：`[苦涩]` (必须为 0)
    *   *说明*：极偏爱水果霉烂后的甜腐味，极其反感苦味。
*   **牛头人 (魔物倾囊制 - 偏好赠予防具)**：
    *   **喜爱风味与期望阈值**：`[麦香: >= 3]` && `[浓郁: >= 2]`
    *   **讨厌风味 (一票否决)**：`[酸爽]` (必须为 0)
    *   *说明*：要求极高浓度的粗犷粮食香与挂杯浓郁感，完全无法接受酸味。
*   **独眼巨人 (魔物倾囊制 - 偏好赠予武器/重矿)**：
    *   **喜爱风味与期望阈值**：`[辣口: >= 3]` && `[温暖: >= 2]`
    *   **讨厌风味 (一票否决)**：`[甜美]` (必须为 0)
    *   *说明*：追求极度火辣与炽热焦温的口感，视人类的甜酒为刷锅水。
*   **幽灵/亡灵 (魔物倾囊制 - 偏好古遗物/秘法卷轴)**：
    *   **喜爱风味与期望阈值**：`[死寂: >= 3]` && `[寒凉: >= 2]`
    *   **讨厌风味 (一票否决)**：`[温暖]` (必须为 0)
    *   *说明*：嗜好死气沉沉与彻骨寒冰的极阴感，拒绝任何温暖。
*   **精灵 (魔物倾囊制 - 偏好稀有作物/法杖)**：
    *   **喜爱风味与期望阈值**：`[香醇: >= 3]` && `[果香: >= 2]`
    *   **讨厌风味 (一票否决)**：`[恶臭]`、`[腐败]` (必须全为 0)
    *   *说明*：追求清雅的花香与果实本味，对恶臭和腐败零容忍。
