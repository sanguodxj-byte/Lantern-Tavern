# 07 - 员工系统

## 1. 招募与定位
随着酒馆规模的扩大，单靠玩家一人白天探索、晚上经营会力不从心。玩家在白天野外探索中，通过特定遭遇或建立信任的剧情事件，可以直接招募其作为员工加入（雇员获取与酒馆等级升级完全脱钩，亦无任何击杀解救设定，完全为自然加入）。酒馆物理升级仅负责为其解锁/增强大堂及工作后勤设施（如矮人熔炉、幽灵文献室），以提供被动增益。

### 1.1 员工的主要职责
每个员工由于其种族天性，拥有完全不同的专属岗位：

| 员工职位 | 种族代表 | 核心功能 | 自动化表现 |
| :--- | :--- | :--- | :--- |
| **酿酒协理** | **精灵酿酒师** | 自动酿造基础酒水。 | 玩家设定好酿造队列后，精灵酿酒师会自动消耗仓库内的材料，批量生产已解锁的普通与优秀级酒谱，无需玩家逐个手动添加。 |
| **大堂服务** | **牛头人跑堂** | 自动接待与秩序维护。 | 晚上自动为落座的顾客端上对应酒水、清理桌子、收集金币；并在有失序症苗头或醉汉闹事时进行武力震慑，确保酒馆安全。 |
| **后勤装备** | **矮人铁匠** | 装备维护与耐久修理。 | 在酒馆后院设立锻造砧，白天为玩家以及来店的人类冒险者修复武器装备耐久，降低装备在地下城折损的概率。 |
| **学术研究** | **幽灵学者** | 古代研究与隐藏配方。 | 白天玩家在外探索时，幽灵学者会在酒馆地下室翻阅残卷，以时间为代价，自动推演并解锁隐藏的传说级酒谱。 |

## 2. 自动化原则与设计界限

为了防止游戏在中后期沦为无聊的“纯放置/数值挂机”游戏，员工系统在设计上遵循严格的**自动化界限**与**用爱发电规则**：

### 2.1 自愿用爱发电规则 (Zero-Wage Agreement)
*   **零薪水与免维护开销**：被招募的员工深深向往酒馆作为地下城唯一和平缓冲区的特殊地位，并对老板抱有绝对的信任。所有招募到店的员工均为**无偿提供劳动力，每日工资为 0，也不消耗任何酒馆资金与材料维护成本**。
*   **无离职与怠工风险**：员工一旦通过剧情被招募，将终身无偿在酒馆内提供专属服务，**绝对不会因为欠薪而产生负面情绪、消极怠工或罢工离职**。这让玩家的夜间经营阶段彻底免除了由于雇人产生的现金流破产红线压力。

### 2.2 员工能做的事 (减少重复劳动)
*   **重复体力活**：批量收取金币、打扫桌椅卫生、给顾客上常规啤酒、自动修理受损防具。
*   **基础消耗酿造**：消耗泛滥材料大批量酿制低级疗伤药酒与招待普通客人的果汁。

### 2.3 员工不能做的事 (保留核心决策)
*   **高级决策权**：决定酒馆是否升级、是否招募新员工、如何重新布置酒馆的家具装潢。
*   **核心社交与委托**：高阶顾客（如独眼巨人王）登门时，必须由玩家亲自上前交谈、接取决定剧情走向的特殊委托；员工无法代理核心谈判。
*   **抵账与赎回处理**：当顾客想要以神兵利器抵扣大额酒账，或者想以新的漂亮亮铁片或材料赎回原有被收下的装备时，必须由玩家亲自在分支选项中拍板决定。
*   **探索策略**：**员工完全不参与白天的野外关卡探索与战斗**。白天的探索与战斗完全由玩家独立进行。员工专注于酒馆内的经营工作，为玩家提供全方位的自动化后勤支持，从而将经营环节彻底转化为玩家的治愈与正反馈时光。

## 3. 员工经营辅助特质系统 (Staff Traits)

为了进一步简化酒馆的夜间经营，被招募的员工留在店中会提供独特的经营辅助效果（被动特质）：
*   **麻利身手 (如哥布林店员)**：清理客桌与收取金币的效率提升 $50\%$。
*   **精打细算 (如矮人铁匠)**：修复装备耐久所需的材料消耗降低 $30\%$。
*   **自然专注 (如精灵酿酒师)**：在精灵协助下，大容量酿酒桶的发酵成功率与发酵状态判定更加稳定。
