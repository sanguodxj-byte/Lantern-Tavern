# 15 - 技能与领悟系统

在《Dungeon Tavern》中，角色没有常规的升级经验与加点树，战斗实力与战术动作完全依赖于**“武器熟练度等级”**和**“主属性面板”**所共同驱动的**技能领悟（获取）系统**。领悟的技能记录在玩家的技能书列表中。

## 1. 双轨技能领悟与释放规则 (Skill Acquisition Rules)

### 1.1 熟练度与主属性双轨领悟阶梯
白天的战斗与探索中，玩家双轨累积武器熟练度与对应的 6 大主属性。当两者同步达到以下数值门槛时，玩家会在战斗或探索中**立刻直接领悟**对应的战斗技能：

*   **【一阶技能 (基础 / Tier 1)】**：
    *   *解锁门槛*：对应武器熟练度等级 $\ge \text{Lv3}$，且该技能主攻属性 $\ge 15$ 点。
*   **【二阶技能 (进阶 / Tier 2)】**：
    *   *解锁门槛*：对应武器熟练度等级 $\ge \text{Lv8}$，且该技能主攻属性 $\ge 35$ 点。
*   **【三阶技能 (绝技 / Tier 3)】**：
    *   *解锁门槛*：对应武器熟练度等级 $\ge \text{Lv15}$，且该技能主攻属性 $\ge 70$ 点。

### 1.2 搭载媒介硬性限制
*   **硬件媒介判定**：为了保证不同武器流派的战术差异，所有已领悟的主动技能必须在玩家**装备对应类型武器**（即释放媒介）时才允许被拉入战斗快捷栏激活使用。
*   **置灰失效**：如果玩家卸下该武器，或者换用其他不匹配的武器类型，快捷栏中的技能会自动置灰失效。例如：装备长弓时无法释放已领悟的双手大剑技能。徒手流派技能只有在双手完全空置（不装备任何武器与防盾）时才允许释放。

### 1.3 ARPG 技能打断与动量继承
当前 3D ARPG 版本中，主动技能不是彼此孤立的按钮，而是可以在动作窗口中互相取消并形成连段。技能运行时需要为每个主动技能记录取消窗口、动量继承比例和动量强化倍率。

*   **取消窗口字段**：
    *   `cancel_start`：技能开始后多少秒允许被其他技能取消。
    *   `cancel_end`：取消窗口结束时间，超过后只能等动作自然结束或被受击打断。
    *   `cancel_into`：允许取消接入的技能类别，如 `movement`、`kick`、`melee`、`block`、`spell`。
*   **动量字段**：
    *   `inherit_momentum`：是否继承当前角色速度和加速度。
    *   `momentum_damage_scale`：有效动量转化为伤害倍率的系数。
    *   `momentum_knockback_scale`：有效动量转化为击退力/推飞距离的系数。
    *   `momentum_cap`：单次技能可吃到的动量强化上限，防止无限叠速。
*   **技能类别建议**：
    *   滑铲、冲撞、战术滑步属于 `movement`，提供高动量，但本身伤害较低。
    *   踢击、盾击、战锤、抓取投掷属于 `impact`，最适合消费动量，把速度转化为击退和地形击杀。
    *   普通近战属于 `melee`，可以吃少量动量伤害，但击退收益低于踢击和战锤。
    *   法术与远程属于 `ranged_or_spell`，通常不继承完整动量，只允许保留释放时的移动惯性。
*   **连段范式**：
    *   **滑铲 + 踢击**：滑铲末段取消接踢击，踢击继承滑铲前冲动量，显著提高击退距离。
    *   **战术滑步 + 精准刺击**：滑步绕侧后取消接刺击，继承少量动量并配合侧击/背袭判定。
    *   **冲撞 + 震荡打击**：冲撞接战锤技能，将接近、碰撞、击退和眩晕压缩成一套高风险爆发。

以下收录了 MVP 版本中的全部 33 个战斗技能。所有技能名称与效果完全采用经典的西方奇幻/中世纪中坚硬核风格（如 *Stoneshard* / *Stardew Valley* 等），数值完全对齐百分比与行动 Tick，无任何武侠或脱节色彩。

## 3. 主属性里程碑技能与通用被动 (Attribute Milestone Skills)

除绑定特定武器媒介的战斗技能外，当角色的 6 大主属性达到特定的数值门槛时，会在战斗或探索中**立刻直接永久领悟**对应的属性被动特权或通用技能。这类技能/被动属于角色自身的身体素养或通用能力，**不需要搭载任何武器媒介，常驻生效或可直接使用**。

### 3.1 属性里程碑解锁规则
*   **【一阶里程碑】**：对应主属性数值 $\ge 5$ 点，满足数值时立刻直接解锁。
*   **【二阶里程碑】**：对应主属性数值 $\ge 15$ 点，满足数值时立刻直接解锁。
*   **【三阶里程碑】**：对应主属性数值 $\ge 30$ 点，满足数值时立刻直接解锁。
