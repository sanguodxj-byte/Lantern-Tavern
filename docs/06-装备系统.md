# 06 - 装备系统

> **文档换算说明（重要）**：本文档原始设计基于格子回合制；当前项目已确认转向 **3D 实时 ARPG**（参考《Barony》式体验）。文中所有"回合 / Tick / 格"均已按以下基准折算为实时表述，具体数值为**设计建议，需按手感调参**：
> - **时间**：1 行动回合 / Tick ≈ 0.5 秒。故"消耗 N 回合动作"→前摇 N×0.5s；"冷却 N 回合"→冷却 N×0.5s；"持续/内 N 回合"→状态持续 N×0.5s。
> - **距离**：技能与位移 **1 格 ≈ 2 米**（地图生成网格单元 = 3.0 米，见 16-技术架构 §6.4）。故"击退 N 格"→击退 N×2 米；"射程/范围 N 格"→N×2 米。
> - 已永久**移除**的格子回合制规则见 [99-深化设计与待办清单.md](99-深化设计与待办清单.md) 的"已取代设计存档"一节（仅作旧设计参考，不直接接入主流程）。
>


为了贯彻 Roguelike 的随机乐趣并确保开发敏捷性，本作装备系统采用**“武器防具品质阶位大表 + 最多 Roll 出 2 个前缀词缀”**的经典硬核战术 Roguelike 规则。装备一经生成，属性即刻固定，无法进行任何强化与洗练。武器装备不再自带技能，而是作为释放对应已领悟技能的搭载媒介。

---

## 1. 装备定位与双轨获取机制

在《Dungeon Tavern》中，装备是玩家白天的核心战斗保障。防具提供被动生存能力与负重，武器则直接决定伤害类型与技能释放权限。

### 1.1 装备获取渠道
*   **野外遗迹搜刮 (常规产出)**：白天在地下城关卡探索时，玩家搜刮**“古代遗迹宝箱”**、**“棺椁陪葬品”**或**“冒险者残骸”**时，能直接获得带有随机词缀的常规装备。野外的普通杂鱼怪物被击杀时，**绝对不会掉落人类装备**，仅掉落酿酒原材料或换钱古玩。
*   **夜晚酒馆消费抵扣与赠予 (极品/盲盒产出)**：
    *   顾客生成时采用**金币/装备互斥双轨携带机制**（详情参见 [12-满意度与价格结算系统.md](12-%E6%BB%A1%E6%84%8F%E5%BA%A6%E4%B8%8E%E4%BB%B7%E6%A0%BC%E7%BB%93%E7%AE%97%E7%B3%BB%E7%BB%9F.md)）。有一半的怪物顾客来店时未携带发光铁片，但随身携带了一件准备喝爽后赠予老板的随机装备。
    *   对人类顾客，因钱不够会拿随身装备抵扣酒钱；对怪物顾客，在喝爽到极点（爆表）且未携带发光铁片时，会狂热地将随身携带的装备赠予老板（系统后台统一计入盲盒抵账）。
    *   **纯随机盲盒机制**：获得（抵扣或赠予）的装备品质、阶位、词缀完全是**全局装备库纯随机抽取**，与当前的势力声望高低无任何概率关联。玩家可能意外得到神级词缀的史诗武器，也可能得到一件没有任何属性的粗糙白板，**甚至可能毫无价值**。

## 3. 战术词缀系统 (Equipment Affix System)

每件装备在生成时，最多可以随机获得 2 个词缀（通过概率分配：没有词缀、1个词缀或2个词缀，2个词缀时必定为 1正1负 以保证战术权衡）。前缀会以定语的形式附加在装备原名前。

### 3.1 6个正向战术词缀 (Positive Affixes)
正向词缀在战斗中提供直接的面板或经验反哺优势：
1.  **锋利的 (Sharp)**：物理/远程伤害额外 $+10\%$，且暴击率 $+3\%$。
2.  **轻盈的 (Lightweight)**：闪避率 $+5\%$，且本装备负重扣减 $30\%$。
3.  **专注的 (Focused)**：攻击命中率 $+10\%$。
4.  **狂暴的 (Furious)**：暴击率 $+8\%$，暴击伤害加成 $+15\%$。
5.  **坚固的 (Sturdy)**：装备物理防御额外 $+4$，且穿着本防具承伤时获得的体质属性经验额外 $+20\%$。
6.  **受洗的 (Blessed)**：法术伤害额外 $+10\%$，或暴击率 $+5\%$。

### 3.2 6个负向战术词缀 (Negative Affixes)
负向词缀表现出装备的粗糙或残损，代偿了极高的属性缺陷：
1.  **生锈的 (Rusty)**：物理/远程伤害降低 $-15\%$。
2.  **笨重的 (Clunky)**：闪避率 $-5\%$，且本装备额外负重增加 $25\%$。
3.  **磨损的 (Worn)**：攻击命中率 $-10\%$。
4.  **劣质的 (Inferior)**：暴击率 $-5\%$，且最大耐久上限降低 $20\%$。
5.  **碎裂的 (Cracked)**：本装备物理防御直接减少 $-3$。
6.  **黯淡的 (Dim)**：法术伤害降低 $-15\%$。

## 4. 装备耐久度与装甲熟练度被动特权

### 4.1 装备耐久度与日常维护
*   **耐久折损**：每一件装备均有固定的最大耐久度。在白天的野外战斗中，每次攻击（如果是武器）或受到击中（如果是防具），均有 $5\%$ 的低概率降低 $1$ 点当前耐久度。
*   **耐久归零**：一旦装备耐久归零，它在后台会被标记为“损坏”，**其提供的所有物理防御、伤害投骰、命中闪避与前缀词缀增益将彻底失效**。但装备不会直接碎裂消失，仍占据背包负重。
*   **铁匠铺修理**：在酒馆营业的夜间，玩家可以将受损装备交给招募的**矮人铁匠**进行修理维护（参见 [07-员工系统.md](07-%E5%91%98%E5%B7%A5%E7%B3%BB%E7%BB%9F.md)）。由于员工为自愿服务的无偿雇员，修理仅需消耗少量的基础木材或铁矿石，免收任何金币手续费。

### 4.2 装甲熟练度阶梯被动特权 (Proficiency Perks Matrix)
为了闭合并对齐 [05-战斗系统.md](05-%E6%88%98%E6%96%97%E7%B3%BB%E7%BB%9F.md)，当玩家通过在野外被击中承伤来累积各防具熟练度等级时，将逐层解锁专属的永久被动加成：

#### A. 轻型装甲熟练度被动 (Light Armor Tiers)
*   **【一阶轻装被动：轻盈步法 / Light Footwork】**：穿着轻型铠甲时，**基础闪避率额外 $+2\%$**。
*   **【二阶轻装被动：侧身闪避 / Sidestep】**：穿着轻型铠甲时，**基础闪避率额外 $+4\%$，且侧击与背击对玩家造成的命中加成降低 $50\%$**。
*   **【三阶轻装被动：疾风闪跃 / Gale Evasion】**：穿着轻型铠甲时，**基础闪避率额外 $+6\%$，受到的暴击率降低 $5\%$，且成功闪避攻击后下一次攻击前摇降低 $15\%$**。

#### B. 重型装甲熟练度被动 (Heavy Armor Tiers)
*   **【一阶重装被动：装甲加固 / Reinforced Armor】**：穿着重型铠甲时，**最终物理防御值额外 $+2$**。
*   **【二阶重装被动：冲击缓冲 / Impact Cushioning】**：穿着重型铠甲时，**最终物理防御值额外 $+5$，且受攻击时有 $20\%$ 概率不损耗装备耐久度**。
*   **【三阶重装被动：不屈壁垒 / Unyielding Bastion】**：穿着重型铠甲时，**最终物理防御值额外 $+10$，完全免疫任何战术击退效果，且承受物理伤害时有 $15\%$ 概率向攻击者反震 $50\%$ 防御值的真实伤害**。
