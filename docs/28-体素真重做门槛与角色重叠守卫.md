# 体素真·重做门槛与角色重叠守卫

> 配套 `docs/17-体素建模工作流.md` §5。本文落实 2026-07-20 对「闪烁 + 假 remake」的工程闸门。

## 1. 问题

1. 体素盒**正体积重叠** → 同材质/共面跨材质 z-fighting（深度闪烁）。
2. 历史玩家与 B/C 批量模型多为 **48px 人形模板 + 贴片/加层**，却被标成 remake/A 档；该批量路径现已禁止。
3. 重叠守卫只覆盖道具，**角色 GLB 管线缺口**。

## 2. 工程闸门

| 闸门 | 位置 | 行为 |
|---|---|---|
| 生成器出口 | `tools/voxel_overlap_guard.py` | `assert_parts_no_positive_volume_overlap` 失败则禁止 export |
| 角色 CI | `tests/gdunit/character_voxel_overlap_test.gd` | 扫 `assets/meshes/characters/voxel_*.glb` 同材质/共面跨材质重叠 |
| 道具 CI | `tests/gdunit/voxel_overlap_scan_test.gd` | 既有 props 扫描 |
| 附着语义 | `voxel_prop_scene_test._boxes_are_attached` | **只认面接触**，正体积重叠不算附着 |
| 文档 | `docs/17` §2 / §5 | 外贴公式 + 真 remake 定义 |

## 3. 真·重做验收清单

发布前全部勾选：

- [ ] 独立 silhouette 像素尺寸表（非 48px 底板换色）
- [ ] 生成器内 overlap guard 全绿
- [ ] `character_voxel_overlap_test` 全绿
- [ ] 真 3D front/side/top/preview（非 ~2KB 结构投影）
- [ ] 质量报告档位与「模板/真 remake」语义一致
