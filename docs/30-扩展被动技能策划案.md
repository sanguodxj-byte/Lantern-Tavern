# 30 - 扩展被动技能策划案（坚韧 / 汲取 / 决斗者 / 奥法之剑）

> **定位**：本文档记录全新扩展的 4 项被动技能设计与机制规范，包含通用恢复/吸血类被动，以及单手剑流派限定的特定动作与魔法联动机制被动。
> 本文档定义将被直接映射并导入 `data/passives.json` 与 `data/skills.json` 配置体系中。

---

## §1 技能策划详表

| 技能名称 | 英文标识 (ID) | 类型分类 | 流派限制 | 详细效果描述 | 触发与失效条件 |
|---|---|---|---|---|---|
| **坚韧** | `tenacity` | 常驻恢复 / Sustain | 无限制（通用） | 每 **5 秒** 自动恢复自身 **2% 最大生命值**。 | 挂载后常驻生效，按 5 秒 Tick 触发治疗。 |
| **汲取** | `siphon` | 战斗吸血 / Combat Lifesteal | 无限制（通用） | 将造成的所有伤害的 **10%** 转化为自身生命恢复（Lifesteal +10%）。 | 造成伤害时触发，按实际结算扣血量的 10% 治疗。 |
| **决斗者** | `duelist` | 战术锁定 / Locked Combat | **单手剑流派限定** (`ONE_HAND_SWORD`) | 准星持续指向单一敌人并**长按右键持续 3 秒**后锁定目标。锁定生效期间，获得仅针对该目标的 **+50% 攻击力**、**+50% 暴击率**、**+50% 闪避率** 加成。 | **准星离开敌人立即失效**；切换目标需重新长按 3 秒锁定。 |
| **奥法之剑** | `spellblade` | 魔法联动 / Cast Charge | **单手剑流派限定** (`ONE_HAND_SWORD`) | 按下**右键**可打开/呼出魔法选择界面；当进行近战**满蓄力攻击**释放时，自动无前摇同时释放选中的魔法。 | 单手剑满蓄力挥刀瞬间触发，消耗对应法力/冷却。 |

## §2 详细机制规范与实现架构

### 2.1 坚韧 (Tenacity)
* **系统层**：由 `SkillRuntime` 或 `Player` 节点的定时器驱动（5 秒周期）。
* **计算公式**：$\text{HealAmount} = \max(1, \text{round}(\text{MaxHP} \times 0.02))$
* **JSON 配置描述**：
  ```json
  "tenacity": {
    "id": "tenacity",
    "name": "坚韧",
    "type": "passive",
    "school": "any",
    "effect_type": "periodic_heal",
    "interval_sec": 5.0,
    "heal_max_hp_pct": 2.0,
    "desc": "每 5 秒恢复 2% 最大生命值。"
  }
  ```

### 2.2 汲取 (Siphon)
* **系统层**：集成于 `DamageResolver` 结算管道与 `CombatBridge`。
* **计算公式**：在阶段五结算扣血时，$\text{lifesteal\_amount} += \text{round}(\text{final\_damage} \times 0.10)$
* **JSON 配置描述**：
  ```json
  "siphon": {
    "id": "siphon",
    "name": "汲取",
    "type": "passive",
    "school": "any",
    "effect_type": "lifesteal",
    "lifesteal_percent": 10.0,
    "desc": "吸取造成伤害的 10% 治愈自身。"
  }
  ```

### 2.3 决斗者 (Duelist)
* **限制条件**：必须装备单手剑 (`ONE_HAND_SWORD`)，非单手剑武器下被动不激活。
* **锁定逻辑与状态机**：
  1. 玩家准星射线检测到敌人 (`Raycast3D` 命中 Enemy)。
  2. 长按右键 (`block` / 聚焦输入) 计时，满 `3.0s` 且准星未离开目标，进入 `DUEL_LOCKED` 锁定状态。
  3. 在 `DUEL_LOCKED` 状态下，针对 `target_enemy` 计算伤害与防御时：
     * `AttackInput.base_damage_bonus_percent += 50.0`
     * `AttackInput.crit_bonus += 50.0`
     * 受击防守时 against `target_enemy`：闪避率加成 `+50.0%`。
  4. 一旦准星偏离目标碰撞体，即刻清除 `DUEL_LOCKED` 状态与一切属性加成。
* **JSON 配置描述**：
  ```json
  "duelist": {
    "id": "duelist",
    "name": "决斗者",
    "type": "passive",
    "school": "ONE_HAND_SWORD",
    "effect_type": "target_focus_lock",
    "lock_hold_sec": 3.0,
    "target_bonus": {
      "damage_pct": 50.0,
      "crit_rate": 50.0,
      "evade_rate": 50.0
    },
    "desc": "单手剑限定。准星指向敌人长按右键 3 秒后锁定对方，锁定后获得仅针对目标的 50% 攻击、50% 暴击、50% 闪避，准星离开敌人立即失效。"
  }
  ```

### 2.4 奥法之剑 (Spellblade)
* **限制条件**：必须装备单手剑 (`ONE_HAND_SWORD`)。
* **输入与技能释放逻辑**：
  1. **右键响应**：单击/长按右键弹出魔法轮盘/菜单选择当前绑定的槽位法术。
  2. **满蓄力检测**：当玩家在近战蓄力 (`charge`) 达到 100% 满蓄力状态并松开攻击时。
  3. **魔法联动**：主手释放单手剑强力斩击的同时，触发选定魔法的无前摇释放。
* **JSON 配置描述**：
  ```json
  "spellblade": {
    "id": "spellblade",
    "name": "奥法之剑",
    "type": "passive",
    "school": "ONE_HAND_SWORD",
    "effect_type": "charge_cast_spell",
    "ui_trigger": "right_click_menu",
    "desc": "单手剑限定。右键可以打开魔法界面，满蓄力攻击的同时会释放魔法。"
  }
  ```

## §3 JSON 数据格式导入规范

这 4 个扩展被动技能在合并至 `data/passives.json` 与 `data/skills.json` 时，将自动兼容项目的配置加载器（Registry），其数值均可由策划直接在 JSON 中灵活调整。
