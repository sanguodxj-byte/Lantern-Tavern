# Domain Docs

Lantern Tavern uses a single domain context.

## Before exploring

- Read `CONTEXT.md` at the repository root when it exists.
- Read relevant decisions under `docs/adr/` when that directory exists.
- If either location is absent, proceed without requiring it to be created.

Use the glossary's terms in issue titles, tests, diagnostics, and architecture
work. Surface any conflict with an existing ADR explicitly rather than silently
overriding it.
