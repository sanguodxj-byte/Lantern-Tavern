# Issue tracker: GitHub

Issues and PRDs for this repository live in GitHub Issues at
`sanguodxj-byte/Lantern-Tavern`.

## Conventions

- Create, read, update, label, and close issues with the `gh` CLI.
- Infer the repository from `git remote -v` when running inside this clone.
- When a skill says to publish to the issue tracker, create a GitHub issue.
- When a skill says to fetch a ticket, include its comments and labels.

The `gh` CLI must be installed and authenticated before a skill can modify
GitHub Issues. If it is unavailable, report that limitation instead of silently
using another tracker.
