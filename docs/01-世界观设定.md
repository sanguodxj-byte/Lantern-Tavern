# 01 - 世界观设定

## 1. 地下城社会
在《Dungeon Tavern》中，地下城并非传统 RPG 里单纯堆满无脑野兽和怪物的巢穴。这里是一个**活着的地下社会**，多个拥有高度智慧的种族在此繁衍生活，其中人类冒险者与部分地下部族在地下城中交错求生。

### 1.1 智慧种族一览
*   **哥布林 (Goblins)**：数量庞大，擅长采集与刺探，通常结成部落，对外界高度戒备，然而对稀缺的生活物资（如美酒、盐）极度渴望。
*   **牛头人 (Minotaur)**：崇尚力量与契约，脾气暴躁却讲究信用，是极佳的护卫与体力劳动者。
*   **独眼巨人 (Cyclops)**：性情古怪，对烈性饮品有超乎寻常的狂热，掌控着地下城深处的稀有矿产与锻造材料。
*   **火蜥蜴 (Salamander)**：生活在火山与熔岩地带，天生带有高热，能提炼特殊的化学药剂与熔岩作物。
*   **幽灵 (Ghost / Phantom)**：地下城遗迹的古老居民，掌握着早已失传的历史文献与炼金术秘密，往往需要在特定精神防护下才能与其交流。
*   **龙裔 (Dragonborn / Dragon)**：地下城的上层统治者或古老领主，拥有极高的尊严与战斗力，对财宝和珍稀的领主级饮品有着近乎偏执的需求。
*   **精灵 (Elves)**：由于历史或信仰原因留居地下城，擅长植物学、精细酿造与魔法学术研究。
*   **人类冒险者 (Human Adventurers)**：从地面涌入地下城的探索者、佣兵和猎人。为了名誉、财富（遗迹宝藏、魔物素材）而来。

## 2. 阵营常态关系与玩家定位

```
┌─────────────────┐           ┌─────────────────┐
│  人类冒险者阵营  │ ◄───────► │  魔物/地下种族  │
│  (掠夺、猎杀)   │  长期敌对 │  (袭击、御敌)   │
└────────┬────────┘           └────────┬────────┘
         │                             │
         └─────────────┬───────────────┘
                       ▼
             ┌──────────────────┐
             │   玩家的酒馆     │
             │ (中立的缓冲区)    │
             └──────────────────┘
```

### 2.1 长期敌对的现实
人类与魔物的敌对并非源于某个“误会”，而是漫长历史掠夺与反抗中形成的血淋淋的现实：
*   冒险者为了金币、装备和地面王国的悬赏，深入地下城猎杀魔物，剥取素材。
*   魔物为了捍卫领地、报复杀戮，设伏并袭击进入地下城的人类。
*   双方都有充足并无法被轻易说服的理由去仇恨彼此。

### 2.2 玩家作为“异类”
玩家并不是试图消灭某一方的英雄，也不是试图建立和平的圣人，玩家只是一个“试图做生意的商人”。
*   **双重不信任**：怪物认为你是个狡猾的人类间谍；人类冒险者认为你是个通敌的叛徒。
*   **行动说话**：玩家无法在对话框里靠油腔滑调解决信任危机，只能通过切实利益交易、提供庇护所、遵守中立承诺来一点点赢取双方的容忍与认可。

## 3. 核心机制：失序症 (Anarchy Syndrome)
地下城中存在着一种名为**失序症**（原狂暴化）的超自然生态灾变，它是游戏主线的重要线索，也是玩家经营的主要阻碍。

### 3.1 失序症的表现形式
失序症不会让生物彻底变成无理性的丧尸，而是会将其**内心最深处的执念极端化、偏执化**：
*   **冒险者失序**：原本追求财富和战胜强敌，失序后转变为疯狂的杀戮欲，无差别攻击视线内的一切活物。
*   **商人/商队失序**：原本追求合理利润，失序后变为疯狂积攒资源，将其筑巢封死，不让任何人碰触。
*   **学者/祭司失序**：原本追求真理与魔法，失序后开始在地下城进行毫无道德底线、极其危险的污染性禁忌实验。
*   **魔物失序**：领地意识与狂暴攻击性被无限放大，无法进行任何交易、接取任务和好感度交流。

### 3.2 应对与解决 (无法打醒与酒馆抑制隐藏设定)
*   **野外遭遇：只能直接击杀**：在野外一旦遭遇处于失序状态的魔物或人类，由于其理智已被执念彻底焚尽，**无法在战斗中通过“打醒”使其恢复**。玩家没有退路，遭遇后必须将其**直接击杀**（或战败/逃跑）。
*   **顾客绝不狂暴**：在酒馆内，**任何已经成为酒馆顾客的普通怪物与任务 NPC 都绝对不会发生狂暴化**。
*   **隐藏设定：酒水抑狂作用**：酒馆销售的各种酒水能够默默抑制失序症的发作。这一设定在游戏初期是**完全隐藏**的。随着游戏流程深入，玩家和任务 NPC 才会从各种口耳相传的闲聊和线索中（如：*“这附近发疯的怪物越来越多了，可天天来你这喝酒的人却一个都没事……”*）慢慢拼凑出这个惊人真相，并推动核心主线剧情的发展。

## 4. 死亡与重生机制 (Death and Rebirth)
地下城对人类与魔物实行完全不同的生死物理规则，这也形成了独特的种族世界观张力：

### 4.1 魔物的“复生与不记仇”法则
*   **魔物无限复生**：地底的智慧魔物（如哥布林、独眼巨人等）死后会根据地底重构规则，于其所属势力的野外巢穴、祭坛或领地中物理重塑复生。当被击杀拯救的失序 NPC 复生重构神体时，其异化执念亦会随之散除净化。
*   **败亡天经地义 (无恶意)**：因为拥有复生的能力，魔物形成了弱肉强食的直白世界观。他们认为战斗失败被玩家杀死是**天经地义**的事，因此**魔物被玩家击杀后，绝对不会对玩家产生任何怨恨、恶意或仇恨标签**。复活后的魔物若在晚上来到酒馆，依然会像往常一样来到酒馆并上涨好感度。
*   **记忆的微量流失**：重生会剥离极少部分死前记忆（主要是为了让任务 NPC 重生时产生轻微的剧情波折），不过不会破坏其与玩家在酒馆建立的友好关系。
*   **人类不可复生**：来自地面的人类（如冒险者）在地下城被杀死，就是**真正的死亡（不可重生）**，生命将彻底终结。
*   **玩家的死亡特权与惩罚**：
    *   作为人类的玩家在探索中被怪物击杀时，受到神秘产权信和地下城契约的特权庇护，灵魂与身体不会彻底消亡。
    *   **死亡代价**：玩家在地下城倒下时，会**失去本次探索背包中收集到的所有材料与道具**，并强行遣送返回酒馆并于柜台后的躺椅上醒来，直接转入下一个经营夜。
    *   **结果**：玩家可以在野外直接、自由地攻击任何魔物以掠夺材料，而不必担心魔物记仇；不过玩家必须在自身状态不佳时警惕被怪物反杀，因为死亡造成的物资清空代价极其沉重。
