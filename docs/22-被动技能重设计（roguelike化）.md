# 22 - 被动技能重设计（传统 Roguelike 化 · NetHack / ToME4 深度）

> **v2 修订说明**：v1 误用了 RoR2 / Isaac / Hades / Dead Cells 作参照——这批是**动作 roguelite（lite）**，
> 核心是数值堆叠 + proc 链 + 爽快感。用户要的是 **NetHack / ToME4 / Angband / DCSS / Caves of Qud / Cogmind**
> 这类**传统 roguelike** 的系统深度。本 v2 推翻 v1 的参照系与被动设计，全部重做。
>
> **适配前提**：本作是**实时动作操控**（hitbox 命中、无回合），所以"传统深度"落在本作的**模拟内核**而非回合——
> 即：状态作为一等公民、sustain（常驻可被驱散）、形态/身体改造、多资源互锁、物品 ego+未鉴定+套装、
> 跨系统涌现、以及**取舍对等**（强必有弱，这是传统 roguelike 的灵魂，lite 没有）。

---

## §0 设计哲学：什么是"like"（传统）而非"lite"（动作 roguelite）

| 维度 | lite（已否决的参照） | like（本作目标 · 传统 roguelike） |
|---|---|---|
| 强度来源 | 数值堆叠（+X% 一堆） | **系统交互涌现**：A 系统状态 × B 系统地形/物品 → 新行为 |
| 被动形态 | 常驻扁平加成 / proc 链 | **sustain（可开启、有代价、可被驱散/沉默）** + 常驻内在(intrinsic) |
| 角色成长 | 数字变大 | **形态/身体改造**（体型、多臂、相位、再生、龙威）——build 定义玩法 |
| 资源 | 单一 stamina / 无 | **多资源互锁**（HP↔法力、正/负能、过载、仇恨），资源间可转化/对冲 |
| 物品 | 白/绿/红分级 +% | **ego 词缀 + 未鉴定 + 套装共鸣**（Angband/ToME4） |
| 核心信条 | 爽 | **取舍对等**：每个强力被动带一个真实弱点（DCSS 种族天性 / ToME4 sustain 代价） |
| 涌现点 | 固定组合 | **动词×环境**（NetHack： grease 护甲→穿刺滑脱；点燃油/蛛网；挖墙；尸食得内在） |

### 重设计的"真深度"五原则（每条里程碑必须命中）

1. **T1 = 入门内在/小 proc**：常驻小效果，常带一个环境/地形交互钩子（NetHack 味）。
2. **T2 = sustain（可开启常驻，有代价，可被驱散/沉默）**：ToME4 式——开启即强，但付出移速/伤害/抗性的真实代价，且敌人可"驱散"它。
3. **T3 = 形态/身体/资源 改变（build-defining，必有取舍）**：Cogmind/Qud 式身体改造、ToME4 式变身、DCSS 式种族天性——改变"你怎么存在"，且强处必伴弱处。
4. **跨系统涌现优先于数值**：被动的效力来自与其他系统（状态/地形/物品/尸体/资源）的交互，而非叠 %。
5. **零回合制残留**：无命中率/投骰/护甲%减免；所有效果基于命中瞬间、状态、情境模拟。

## §1 A 组重设计：18 项属性里程碑（ToME4 天赋树式 · 带取舍）

每属性 = 一棵 mini 天赋树：T1 入门节点 / T2 sustain / T3 形态-改变。标注 **取舍** 与 **经典类比**。

### 力量 STR — 蛮力 / 形态

**① 震荡冲击** `STR T1`
- 机制：近战命中 15% 击退 1m；**撞墙追加 `STR×0.5` 碰撞伤害**（地形交互）。常驻：负重上限 +10。
- 经典：NetHack「把敌人撞进墙/水」的地形交互；被动无副作用。
- 落地：复用 `ME.try_knockback_chance`（加撞墙伤害分支）。

**② 磐石姿态** `STR T2` · **[sustain]**
- 机制：开启后**站桩/格挡时减伤 +20% 且免疫击退**；代价：**移速 -35%、翻滚无敌帧失效**。可被"驱散/沉默"解除。
- 经典：ToME4 sustain（如 Stone Skin / Aegis）——开启即强，代价明确，可被打断。
- 落地：新 `ME.sustain_bulwark`（需 sustain 框架 + 驱散接口）。

**③ 巨化** `STR T3` · **[形态]**
- 机制：体型变大——近战范围 +0.6m、伤害 +15%、可撞开轻甲敌；**取舍**：无法通过低矮通道、**翻滚无敌帧失效**、**被命中体积更大（更易被 AoE/范围命中）**。
- 经典：Cogmind「换大框架」/ Qud「体型 mutation」——身体改造改变玩法且有真实弱点。
- 落地：新 `ME.form_giant`（改 hitbox 体积 + 通道判定 + 范围命中权重）。

### 敏捷 DEX — 机动 / 潜行 / 多臂

**④ 翻滚突进** `DEX T1`
- 机制：翻滚 1.5m + 0.3s 无敌帧（取代纯跳跃）；翻滚后 2s 进入**潜行**（敌人感知半径 -40%）。
- 经典：NetHack `stealth` 内在 / ToME4 潜行；动作壳 + 传统潜行状态。
- 落地：active 路径（带 i-frame）+ `set_stealth` 状态。

**⑤ 疾矢** `DEX T2` · **[sustain]**
- 机制：远程武器常驻 +15% 攻速、**-10% 伤害**（代价）；箭矢/弩矢**可回收**（命中落地可拾回，NetHack 弹药复用）。
- 经典：ToME4 sustain 取舍 + NetHack「可回收弹药」资源节俭。
- 落地：新 `ME.sustain_swift_aim` + 弹丸可回收标志。

**⑥ 千手** `DEX T3` · **[身体改造]**
- 机制：获得**第二条持械臂**，可同时持一件副手近战/盾；**取舍**：副手武器须轻量化（重武器冲突，双持重武 disabled）。
- 经典：Cogmind/Qud「多臂 body-mod」——改变装备槽结构，带约束。
- 落地：新 `ME.body_extra_arm`（改装备槽校验 + 武器重量门控）。

### 灵巧 AGI — 闪避 / 残影 / 相位

**⑦ 残影闪** `AGI T1`
- 机制：受近战 10% 侧闪免伤；闪避后 2s 下次攻击 +20%（proc + 条件）。
- 经典：传统闪避内在 + 反击窗口。
- 落地：扩展 `ME.try_sidestep`。

**⑧ 鬼影步** `AGI T2` · **[sustain]**
- 机制：常驻 10% 闪避；代价：**受击时 15% 概率自身陷入"残影混乱"（0.4s 失控、无法操作）**（sustain 的真实风险）。移动留下残影分身吸引仇恨，但本体伤害 -10%。
- 经典：ToME4 sustain 风险（如某些 sustain 有反噬）+ 分身 aggro 模拟。
- 落地：新 `ME.sustain_phase_step`（需受击事件 + 混乱状态）。

**⑨ 相位** `AGI T3` · **[内在/形态]**
- 机制：每隔 12s 自动短距闪现脱离包围；受致死伤 25% 相位规避。**取舍**：**你无法被治疗/护盾类效果作用于（相位排斥增益）**。
- 经典：DCSS 不死族内在（undead 不被治疗）/ ToME4 Ethereal Form  intangible 但有条件。
- 落地：新 `ME.intrinsic_phase`（闪现计时 + 增益屏蔽标记）。

### 体质 CON — 减伤 / 再生 / 不死

**⑩ 铁血** `CON T1`
- 机制：最大 HP +20；每损失 10% 当前 HP，受击最终扣血 -1（阶梯减伤，保底 1）。
- 经典：Angband/DCSS 种族减伤内在。
- 落地：扩展 `ME.apply_thick_skin` 为阶梯。

**⑪ 再生** `CON T2` · **[sustain]**
- 机制：常驻缓回（每秒 1.5% 最大 HP）；**取舍**：饱食度/能量消耗加快，且**中毒时再生反转（转为掉血）**——状态交互（NetHack 中毒/饥饿）。
- 经典：NetHack 再生 vs 中毒/饥饿的模拟冲突；sustain 代价。
- 落地：新 `ME.sustain_regen`（需饥饿/中毒状态监听）。

**⑫ 不朽** `CON T3` · **[形态/资源]**
- 机制：每场战斗首次致死伤 → 20% HP 存活一次；**取舍**：复活后 5s 全属性 -20%（虚弱）。
- 经典：ToME4 "unstoppable"/DCSS Felid 九命——每战一次保命，带惩罚窗口。
- 落地：新 `ME.form_undying`（战斗范围一次性标记 + 虚弱 debuff）。

### 魔力 MAG — 法术 / 双资源 / 超载

**⑬ 法力回响** `MAG T1`
- 机制：法术 10% CD 清零；清零后下次该技能 +30%。
- 经典：传统冷却/增益链。
- 落地：扩展 `ME.try_mana_focus_cd_reset`。

**⑭ 元素亲和** `MAG T2` · **[资源/状态]**
- 机制：你拥有**正/负双能池**，法术消耗可在两池间切换；对**已处元素异常**的敌人法术 +20%。
- 经典：ToME4 正/负能（Anorithil）+ 状态增幅。
- 落地：新 `ME.dual_energy`（需双资源框架 + status 读取）。

**⑮ 奥能超载** `MAG T3` · **[资源转化/形态]**
- 机制：法术暴击 +8%，暴击触发小范围元素迸发 AoE；**取舍/资源**：可**消耗当前 HP 施放"无消耗法术"（HP↔法力 转化）**——过载有自伤风险。
- 经典：ToME4 vim/paradox 资源对冲（用血换法力/用时间换力）。
- 落地：扩展 `ME.apply_mana_surge_crit` + 新 `ME.overload_hp_cost`（HP→法力转化）。

### 感知 PER — 暴击 / 警觉 / 预知

**⑯ 猎手直觉** `PER T1`
- 机制：视野 +1m、陷阱发现 +20%；对**未发现你**的敌人首击必暴（潜行偷袭）。
- 经典：NetHack `warning` / `stealth` 内在（知道危险/不被发现）。
- 落地：复用 `ME.alertness_bonus` + 首击必暴（需 `target_aware`）。

**⑰ 破绽洞悉** `PER T2`
- 机制：基础暴击 +5%；暴击施加**破甲标记**（目标后续受击 +8%）。
- 经典：Angband/ToME4 vulnerable 标记。
- 落地：扩展 `ME.crit_rate_bonus` + 新 `ME.apply_vulnerable_mark`。

**⑱ 预知** `PER T3` · **[内在 + 取舍]**
- 机制：免疫侧击/背袭加成；受击前 0.4s 25% 自动闪避；**常驻"警觉"——相邻高威胁/埋伏提示**。
- **取舍**：感知过高使你**更易被灵能/精神系敌人锁定**（ToME4 psi/paradox backlash 式反噬）。
- 经典：NetHack `warning` 内在 + DCSS 高感知识别的代价。
- 落地：扩展 `ME.negate_flank_bonus` + 新 `ME.premonition` + 灵能锁定标记。

## §2 Duo 协同层（跨系统涌现 · NetHack 式"这 + 那 = 新行为"）

> 不是"两件套 +X%"，而是**两个系统的状态/形态碰撞出第三行为**。需同时拥有指定里程碑。

| Duo | 需要 | 涌现效果 | 经典类比 |
|---|---|---|---|
| 巨力双臂 | 巨化(STR T3) × 千手(DEX T3) | 体型变大 + 第二持械臂 → **可双持两把双手武器**（身体×身体，build-defining） | Cogmind/Qud 身体组合 |
| 血法循环 | 不朽(CON T3) × 奥能超载(MAG T3) | 超载用 HP 施法时，若 HP 跌破阈值**自动触发不朽**保命（资源→生存 链） | ToME4 资源对冲链 |
| 相位预判 | 相位(AGI T3) × 预知(PER T3) | 预知警觉预警 → **自动触发相位闪现**（免伤链） | 状态×内在 涌现 |
| 磐石铁壁 | 磐石姿态(STR T2) × 铁血(CON T1) | 站桩减伤叠加阶梯减伤；但**移速惩罚同步叠加**（取舍叠加，非纯增益） | sustain 代价叠加 |
| 燃矢 | 疾矢(DEX T2) × 元素亲和(MAG T2) | 回收箭矢**附带元素附着**（箭→元素弹，物品×状态） | NetHack 物品×状态 |
| 破墙 | 破绽洞悉(PER T2) × 震荡冲击(STR T1) | 破甲标记敌人被击退**撞墙伤害 ×2**（状态×地形） | NetHack 地形×状态 |

## §3 C 组重设计：武器/盾 → 传统「ego 物品 + 未鉴定 + 套装」层

> 取代 v1 的 RoR2 白绿红分级。改为 **Angband/ToME4 式 ego 词缀 + 未鉴定 + 套装共鸣 + 武器精通树**。

- **ego 词缀**（非固定数值）：每把武器/盾生成时带 1–2 个 ego，如
  - `of Flames` 命中点燃 / `Frost` 命中减速 / `Vampiric` 吸血 / `of Slaying <族>` 对某类 +50% /
  - `Flaming Frost`（双元素冲突 ego，NetHack 式稀有矛盾）/ `Keen`（暴击+）/ `Lance`（穿刺+）。
- **未鉴定系统**：拾取只显外观/重量/手感；**使用/鉴定卷轴后才揭示 ego**（NetHack/Angband 鉴定深度——信息即权力）。
- **套装共鸣（set）**：集齐 2/3 件触发套装效果（如「龙威」套 → 龙吼光环恐惧周围敌）。→ ToME4 set items。
- **武器精通树**：该类被动 = 你对该类武器的**精通天赋树**，解锁条件不是"拿武器"而是"用该类击杀 N 敌 / 熟练度"（ToME4 mastery）；树内 T1/T2/T3 同样遵循 §0 五原则（含 sustain/形态）。
- **形态门控 ego**：某些 ego 仅对你有特定形态生效（如「双持 ego」需千手；「巨型 ego」需巨化）——物品×身体 涌现。

12 条武器线（单手剑/战锤/长枪/双手大剑/双手斧/长弓/轻弩/投掷/法杖/魔导书/徒手/盾）各自一棵精通树，
每树 T1 入门内在 / T2 sustain(取舍) / T3 形变（如：长弓 T3「分裂三矢」、盾 T3「格挡生盾」、徒手 T3「化劲反弹」）。

## §4 B 组机制被动（深化为传统风 · 加取舍/sustain）

- `cancel_recovery` → 取消后摇接翻滚带 0.2s 无敌帧（动作壳），但**翻滚后 0.5s 攻速 -15%**（取舍）。
- `ram`（撞敌增强）→ 撞敌附加「短暂破甲标记」，与 PER T2 破绽洞悉**跨组协同**（接 Duo 破墙）。
- `throw_recover` → 投出后 4s 内召回，**召回途中穿透敌人**（接 C 组投掷 T3 形变）。
- `auto_loot` → 脱战时自动拾取 8m 内掉落；**但自动拾取不触发鉴定**（信息仍隐蔽，NetHack 味）。

## §5 实现层：传统深度所需的模拟内核（前置 hook）

v1 的 `CombatEventBus`（on_hit/on_dodge/on_kill…）仍是基础，但传统深度需要更重的**模拟层**：

| 子系统 | 说明 | 消费方 |
|---|---|---|
| **状态模拟层** | 一等公民状态：blind/confuse/stun/slow/burn/poison/freeze/vulnerable/stealth/intangible/bleed/fear，各有成因与解除 | 元素亲和、破绽洞悉、鬼影步、再生 |
| **sustain 框架** | 常驻效果容器，支持开启/关闭/**被驱散/沉默** | 磐石姿态、疾矢、鬼影步、再生 |
| **身体/形态槽** | 体型、持械臂数、相位、不朽——改变 hitbox/装备槽/被增益规则 | 巨化、千手、相位、不朽、奥能超载 |
| **多资源系统** | HP / 法力 / 正能 / 负能 互转与对冲 | 元素亲和、奥能超载、血法循环 Duo |
| **物品 ego + 鉴定 + 套装** | 生成时挂 ego、未鉴定态、套装计数 | C 组全部 |
| **地形/环境交互** | 撞墙判定、通道高度、可点燃物、尸体 | 震荡冲击、巨化、燃矢、NetHack 式涌现 |
| **驱散/沉默接口** | 敌人可剥夺你的 sustain/内在 | 所有 sustain 的代价闭环 |
| 沿用 §1 标注的 `on_hit/on_dodge/on_take_damage/target_aware/hp_ratio` | 事件总线 | proc 层 |

## §6 平衡（传统信条：取舍对等）

- **每个 T2 sustain 必须有一个可被敌人利用的代价**（移速/伤害/失控/反噬），且可被驱散。
- **每个 T3 形态必须有明确弱点**（体积/无法被治/虚弱窗口/装备冲突）。
- **proc 概率**：T1 10–15%；T2/T3 形变 100%（必触发，因为是玩法改变，不靠概率爽）。
- **Duo 强度 ≈ 单 T3 的 60–80%**，且 Duo 是"涌现"而非"加算"。
- **资源转化必有风险**（HP 换法力会自伤；双能切换有冷却）。

## §7 落地顺序建议

1. **先建模拟内核**（§5 状态层 + sustain 框架 + 事件总线）——没有它，T2/T3 无从落地。
2. **A 组 T1（无新系统，纯内在/proc）** 可立即落，复用现有 `ME` 函数。
3. **A 组 T2 sustain**（需 sustain 框架）。
4. **A 组 T3 形态**（需身体/形态槽 + 资源系统）。
5. **Duo 层**（集合运算 + 跨系统读取）。
6. **最后 C 组**（ego/鉴定/套装/精通树，依赖整套模拟内核，单独排期）。
