# 31 - 握持流派与大类武器核心机制纯被动设计方案（最新精细修正）

> **设计核心原则**：
> 1. **严禁出现击退相关效果**（全面删除任何 Knockback 击退、冲量及相关描述）。
> 2. **拒绝简单数值拉条**（摒弃简单的 +10% 攻速/防具提升等无趣味纯数值）。
> 3. **全量为深度核心机制**（对齐“决斗者”与“奥法之剑”，重点设计操作触发、战术锁定、动作联动、状态转换与环境反制）。
> 4. **键位与姿态严密相容**（例如右键在非单手状态下被格挡占用，故“奥法之剑”归属于【单手流派】）。
> 
> $$\text{Build 最终被动机制} = \text{握持流派核心机制 (Style Mechanics)} + \text{武器大类核心机制 (Weapon Mechanics)}$$
