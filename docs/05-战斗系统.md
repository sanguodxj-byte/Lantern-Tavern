# 05 - 战斗系统

## 1. 战斗定位与基本模式

### 1.1 战斗定位
在《Dungeon Tavern》中，击杀魔物并不会带来高额的金币掉落或直接的经验升级。**战斗并非赚钱的主要手段**。
战斗的作用定位为：
*   **野外自保**：在搜集酿酒材料的过程中，摆脱主动袭击的野生魔物或失序者。
*   **获取认可**：在地下城崇尚力量的种族中，击败对手是获得其尊重与对话权的最快途径。
*   **解决冲突与维持秩序**：通过武力将失序（狂暴化）的 NPC 打醒，拯救他们并重新打通受阻的商路。
*   **主动攻击与击杀无害**：
    *   **主动攻击权**：玩家拥有绝对的自由在野外直接发动先手攻击，猎杀任何遇到的魔物来掠夺其素材。
    *   **不记血债与怪物复生**：由于魔物拥有地底复生特权，并崇尚力量至上，他们认为在野外战斗失败被杀是天经地义的自然法则。因此**被击杀不会在魔物心中产生任何仇恨或负面印象**。这保证了玩家在地下城探索时可以毫无心理负担地砍怪，避免导致酒馆生意无法进行。

### 1.2 玩家中心回合制 (Player-Centric Turn-Based)
系统参考了传统经典 Roguelike 游戏（如 *Stoneshard*）的战斗模式：
*   **同步更新**：游戏世界以玩家的动作为核心。只有当玩家移动一格、使用道具、发动攻击或等待一回合时，地下城的所有怪物和环境机制才会同步行动。
*   **策略思考**：没有时间流逝的压力，玩家可以有充分的时间观察环境、计算距离，规划每一步的技能和走位。
*   **移动行动 Tick 与负重限制 (Movement & Encumbrance Costs)**：
    *   **正常负重 (负重 $\le 70\%$ 上限)**：玩家每次移动一格消耗 **$1$ 行动 Tick**。
    *   **重荷状态 (负重 $70\% - 100\%$ 上限)**：移动力受到负累，每次移动一格被动消耗 **$2$ 行动 Tick**（即玩家走一步，周围怪物可以行动两步）。
    *   **超重状态 (负重 $> 100\%$ 上限)**：移动力强行归 0，无法主动通过移动改变位置。

## 2. 6大属性与战斗面板换算公式

玩家在野外战斗与探索的实力，完全由 6 大主属性决定。主属性通过实战使用武器积累熟练度，以及在战斗中执行对应动作获取属性经验来进行提升。

### 2.1 基础属性与战斗面板换算
*   **最大生命值 (Max HP)**：由体质与角色等级共同主导。
    $$\text{Max HP} = 100 + \text{体质} \times 10 + \text{角色等级} \times 5$$
*   **物理防御力 (Physical DEF)**：由装备防御与体质共同决定。
    $$\text{最终物理防御} = \text{防具物理防御值} + \text{体质} \times 1$$
*   **负重上限 (Carry Weight Limit)**：由体质决定，直接影响探索背包容积。
    $$\text{负重上限} = 50 + \text{体质} \times 2$$
*   **近战物理伤害修正 (Melee Flat Bonus)**：由力量决定，影响单手剑、双手大剑、双手手、战锤、长枪、匕首等近战伤害修正。
    $$\text{近战最终伤害修正} = \text{力量 (STR)} \times 1.5 + \text{战斗风格与熟练度修正}$$
*   **远程物理伤害修正 (Ranged Flat Bonus)**：由敏捷决定，影响长弓、轻弩等远程伤害修正。
    $$\text{远程最终伤害修正} = \text{敏捷 (DEX)} \times 1.5 + \text{武器与熟练度修正}$$
*   **法术伤害修正 (Spell Flat Bonus)**：由魔力决定，影响附魔法杖、魔导书等魔法伤害修正。
    $$\text{法术最终伤害修正} = \text{魔力 (MAG)} \times 1.5 + \text{武器与熟练度修正}$$
*   **闪避率 (Evade Rate)**：由灵巧与装备共同决定，直接以百分比形式减少攻击者的命中率。
    $$\text{最终闪避率}(\%) = \text{装甲/盾牌基础闪避加成}(\%) + \text{灵巧} \times 1\%$$

## 3. 战斗风格与武器装备穿戴限制 (Combat Styles)

战斗中，根据玩家当前的装备手部占用状态，自动激活以下战斗风格。双手武器从规则上绝对禁止用于双持：

*   **【单手风格】**：主手装备 1 把单手近战武器（单手剑、战锤、匕首），副手空置。
    *   *战术定位*：决斗者。
    *   *数值修正*：物理攻击命中率 $+10\%$，闪避率 $+5\%$。
    *   *风格专属技能*：`【决斗挑战 / Duelist Challenge】`。
*   **【单手持盾风格】**：主手装备 1 把单手近战武器，副手装备 1 把防盾。
    *   *战术定位*：守护者。
    *   *数值修正*：激活盾牌格挡判定，物理防御防守成功时，最终物理伤害值固定降低 $15\%$。
    *   *风格专属技能*：`【护卫誓言 / Oath of Guardian】`。
*   **【双手风格】**：双手持用 1 把双手近战/法系武器（双手大剑、双手斧、战锤、长枪、附魔法杖）。**副手绝对禁止同时装备任何防盾或副手武器**。
    *   *战术定位*：破坏者。
    *   *数值修正*：造成的物理/法术基础伤害乘以双手专属基础倍率，同时击退判定距离额外 $+1$ 格。代偿代价为闪避率降低 $10\%$。
    *   *风格专属技能*：`【毁灭重击 / Devastating Blow】`。
*   **【双持风格】**：主手与副手各持 1 把单手近战武器（单手剑、匕首）。**绝对不能双持任何双手武器（如双手大剑、双手斧）**。
    *   *战术定位*：狂战士。
    *   *数值修正*：单回合可自动发动 2 次攻击判定（主手造成 $100\%$ 伤害，副手造成 $60\%$ 伤害）。代偿代价为每次物理攻击的命中率降低 $10\%$。
    *   *风格专属技能*：`【双刃连击 / Dual Strike】`。
*   **【徒手风格】**：双手均完全空置，不装备任何武器。
    *   *战术定位*：武术家。
    *   *数值修正*：伤害与百分比命中修正由力量与灵巧数值相加主导。攻击所消耗的行动 Tick 减少 $30\%$。
    *   *风格专属技能*：`【铁拳 / Iron Fist】`。

## 4. 伤害结算与属性倍率加算算法 (Damage Resolution Algorithm)

战斗中的单次伤害判定，由系统按照以下 6 个精细阶段构成的算法流程进行依次执行与结算：

### 4.1 阶段一：命中与闪避判定 (Hit & Evade Check)
系统进行以下百分比攻守对抗判定：
$$\text{最终命中率 } P_{hit}(\%) = \text{基础命中率 } (75\%) + \text{攻方命中率加成}(\%) - \text{防方闪避率}(\%)$$
*   **攻方近战命中率加成** = 武器基础命中加成(%) + 力量 $\times 0.5\%$ + 感知 $\times 0.5\%$ + 战斗风格加成(%)。
*   **攻方远程命中率加成** = 武器基础命中加成(%) + 敏捷 $\times 1\%$ + 战斗风格加成(%)。
*   **防方闪避率(%)** = 装甲/盾牌基础闪避率(%) + 灵巧 $\times 1\%$ + 战斗风格加成(%)。
*   **判定结果**：
    *   **限制边界**：最终命中率 $P_{hit}$ 最低下限锁定为 $5\%$，最高上限锁定为 $95\%$，保留基本概率博弈。
    *   **判定执行**：系统随机生成一个 $[1, 100]$ 的整数 $D_{roll}$。
    *   **击中**：若 $D_{roll} \le P_{hit}$，则攻击命中，进入阶段二。
    *   **未击减/闪避**：若 $D_{roll} > P_{hit}$，则攻击未命中，本次攻击结束，防方结算闪避相关的动作双轨经验。

### 4.2 阶段二：暴击判定 (Critical Hit Check)
如果命中，系统进行以下暴击率对抗判定：
*   **攻方最终暴击率** = $5\% + \text{攻方感知} \times 0.5\% + \text{装备/技能修正}$。
*   **最终判定暴击率** = 攻方最终暴击率 - 防方感知 $\times 0.5\%$ (最低降至 $0\%$)。
*   **判定执行**：系统随机生成一个 $[1, 100]$ 的整数，若该点数小于或等于最终判定暴击率，则本次攻击成功触发暴击。进入阶段三时暴击标记设为 True。

### 4.3 阶段三：风格与武器基础伤害计算 (主属性倍率加算)
根据攻方的伤害类型与武器基底，计算未减免的前置基础伤害值。伤害公式如下：
$$\text{基础伤害} = (\text{武器伤害投骰 NdN} + \text{伤害修正}) \times \text{最终伤害倍率}$$
其中，主攻属性和熟练度作为加数，直接对最终伤害倍率进行加算增幅，以遏制二次乘数膨胀：

#### A. 近战伤害类型（双手、单手、持盾、双持、徒手）
*   **武器投骰 NdN**：由装备的近战武器决定（徒手攻击为固定的 `2D10`）。
*   **物理伤害修正**：
    *   双手风格：力量 (STR) $\times 1.5 + \text{双手熟练度} \times 0.5$。
    *   单手风格：力量 (STR) $\times 1.0 + \text{灵巧 (AGI)} \times 0.2 + \text{单手熟练度} \times 0.2$。
    *   持盾风格：力量 (STR) $\times 0.5 + \text{体质 (CON)} \times 0.3 + \text{防盾熟练度} \times 0.1$。
    *   双持主手：力量 (STR) $\times 0.8 + \text{灵巧 (AGI)} \times 0.4 + \text{双持熟练度} \times 0.1$。
    *   双持副手：主手伤害修正 $\times 0.5$。
    *   徒手风格：（力量 (STR) + 灵巧 (AGI)） $\times 1.0 + \text{徒手熟练度} \times 0.3$。
*   **最终物理伤害倍率 (加算公式)**：
    *   双手风格：$1.0 + \text{双手熟练度} \times 0.005 + \text{力量 (STR)} \times 0.015$。
    *   单手风格：$1.0 + \text{单手熟练度} \times 0.003 + \text{灵巧 (AGI)} \times 0.01$。
    *   持盾风格：$0.85 + \text{防盾熟练度} \times 0.002 + \text{力量 (STR)} \times 0.005$。
    *   双持主手：$1.0 + \text{双持熟练度} \times 0.005 + \text{力量 (STR)} \times 0.008$。
    *   双持副手：$0.6 + \text{双持熟练度} \times 0.003 + \text{力量 (STR)} \times 0.005$。
    *   徒手风格：$1.0 + \text{徒手熟练度} \times 0.005 + \text{力量 (STR)} \times 0.01$。

#### B. 远程伤害类型 (长弓、轻弩)
*   **武器投骰 NdN**：由装备的长弓/轻弩决定。
*   **远程伤害修正** = 敏捷 (DEX) $\times 1.5 + \text{远程熟练度} \times 0.4$。
*   **最终远程伤害倍率** = $1.0 + \text{远程熟练度} \times 0.005 + \text{敏捷 (DEX)} \times 0.01$。

#### C. 法术伤害类型 (附魔法杖、魔导书)
*   **武器投骰 NdN**：由装备的法术媒介决定。
*   **法术伤害修正** = 魔力 (MAG) $\times 1.5 + \text{魔法熟练度} \times 0.4$。
*   **最终法术伤害倍率** = $1.0 + \text{魔法熟练度} \times 0.005 + \text{魔力 (MAG)} \times 0.01$。
*   **暴击增幅**：若暴击标记为 True，算出的基础伤害直接乘以最终暴击倍率：
    $$\text{最终暴击倍率} = 150\% + \text{攻方感知} \times 1\% - \text{防方感知} \times 1\% \quad (\text{最低为 } 110\%)$$

### 4.4 阶段四：朝向判定最终修正
依据攻防双方在网格中的朝向关系，进行最终的命中与伤害调整：
*   **侧击**：攻方命中率加成额外 $+5\%$。
*   **背袭**：防方格挡率强行清零（无法用盾牌格挡），同时本次基础伤害值直接乘以 $1.5$。

### 4.5 阶段五：防御力与格挡减免结算
计算防方的最终护甲抵免值：
$$\text{最终物理/法术防御} = \text{防具物理/法术防御值} + \text{体质} \times 1$$
*   **基础抵免**：伤害值直接减去最终防御力，受到的伤害最少为 1 点。
*   **盾牌格挡判定**：若防方手持盾牌，投掷 1D100 对抗防方最终格挡率。
    *   **格挡成功**：伤害值额外降低，降低数值等于盾牌提供的格挡值。
    *   **风格联手**：若激活了单手持盾风格，格挡成功后受到的最终物理伤害再固定降低 $15\%$。

### 4.6 阶段六：最终扣血与战术位移结算
*   **生命扣除**：将阶段五算出的最终伤害数值，从防方的当前生命值 (HP) 中扣除。
*   **战术效果附加与专属大招判定**：
    *   **击退判定**：造成伤害的重型武器触发击退。目标被往后击退 1 格（双手风格击退 2 格）。目标被击退方向存在墙壁或障碍物时，受到等同于该次物理攻击伤害 $30\%$ 的额外碰撞伤害并眩晕 1 回合（战锤为 $60\%$ 碰撞伤害与眩晕 2 回合）。
    *   **决斗挑战效果**：单手风格下释放招牌大招`【决斗挑战】`时，锁定 1 名敌人。在 3 个行动回合内，玩家面向该目标时命中率额外 $+30\%$，伤害额外 $+30\%$，同时成功闪避其近战物理攻击时，自动对该目标进行 1 次 $100\%$ 近战物理伤害的免费反击。技能 CD 10 回合。

### 4.7 ARPG 地形伤害与物理位移联动

当前 3D ARPG 版本中，战术位移不只服务于控制距离，也应成为地下城战斗的核心击杀手段。踢击、冲撞、抓取投掷、战锤击退、双手武器重击等效果都可以把敌人推入危险地形，从而触发尖刺、酸液、坠落、火焰等额外伤害。

*   **踢击处决示例**：玩家面对靠近尖刺陷阱的敌人时，可以用踢击将其推入尖刺区域。若敌人碰撞后进入 `SpikesTrap`，立即触发地形伤害；低生命或普通小怪可被尖刺直接击杀。
*   **地形伤害归属**：由玩家主动位移造成的地形伤害，应计为玩家战术结果，用于触发击杀反馈、材料掉落、熟练度/属性经验或相关成就统计。
*   **位移优先级**：击退结算时先计算攻击命中和基础伤害，再施加位移冲量；目标移动过程中若触碰危险 Area3D，则结算地形伤害；若撞上墙体或障碍物，则结算碰撞伤害与短时硬直。
*   **危险地形类型**：
    *   **尖刺**：高额瞬间物理伤害，适合作为踢击、冲撞、抓取投掷的处决目标。
    *   **酸液**：持续腐蚀或短时高频伤害，适合配合减速、击退和封路。
    *   **坠落 / 深坑**：高风险即死或重伤地形，只能在有清晰视觉提示和替代路径的区域生成。
    *   **墙体 / 柱子 / 石堆**：不一定造成地形伤害，但可承接击退碰撞，触发额外碰撞伤害、眩晕或破防。
*   **反向风险**：敌人也可以利用位移把玩家逼向危险地形。高阶敌人的冲撞、盾击或抓取必须有明显前摇，给玩家闪避、格挡或反向利用地形的机会。
*   **战斗空间要求**：地图生成应围绕危险地形提供可读的踢击路线、绕后路线和安全站位，使“把敌人踢向尖刺”成为玩家自然发现的技巧，而不是偶然发生的物理漏洞。

### 4.8 技能打断、动量继承与加速度强化 (Cancel & Momentum Chain)

当前 3D ARPG 战斗采用实时动作反馈，主动技能之间可以在特定窗口内互相打断，并把角色当前速度、加速度和位移方向继承到下一个技能中。该机制用于鼓励玩家把移动技能、控制技能和地形伤害组合起来，例如“滑铲 → 踢击”可以借滑铲的前冲速度把敌人踢飞更远，甚至直接送入尖刺陷阱。

*   **技能取消窗口**：
    *   每个主动技能可配置 `cancel_start` 与 `cancel_end`，只有进入取消窗口后才能被下一个技能打断。
    *   防御姿态、结界等站桩型技能通常只能被闪避、格挡或受击打断；滑铲、冲撞、战术滑步等位移技能允许更早取消。
    *   受击、格挡成功、地形碰撞也可以强制打断当前技能，但不一定保留玩家主动连段的动量奖励。
*   **动量继承规则**：
    *   角色在释放下一个技能时读取上一帧的水平速度、加速度和移动方向，写入 `MomentumContext`。
    *   若新技能方向与当前动量方向夹角小于 45 度，继承大部分动量；45-90 度只继承少量；超过 90 度视为反向刹车，不获得强化。
    *   动量继承只保留水平分量，避免跳跃、台阶、掉落导致异常爆发。
*   **伤害与位移强化**：
    *   动量强度越高，下一次命中的 `damage_mult`、`knockback_force`、`dash_distance` 或目标被推飞距离越高。
    *   推荐基准：每 1.0 米/秒有效动量提供约 `+3%` 伤害与 `+8%` 击退距离；强化上限默认锁定在 `+30%` 伤害与 `+80%` 击退距离。
    *   踢击、盾击、冲撞、抓取投掷、战锤重击最适合吃动量加成；法术、远程射击只继承少量位移安全性，不获得完整击退强化。
*   **连段示例**：
    *   **滑铲 → 踢击**：滑铲末段取消接踢击，踢击继承前冲动量，敌人获得额外击退力，可被踢入更远处的尖刺或酸液。
    *   **冲撞 → 战锤震荡打击**：冲撞接近并打断为战锤攻击，若命中墙边敌人，碰撞伤害与眩晕更稳定触发。
    *   **战术滑步 → 背袭刺击**：侧向滑步绕到敌人侧后方后取消接刺击，继承少量动量并获得朝向优势。
*   **平衡限制**：
    *   动量强化每次连段后衰减，禁止无限滑步叠加速度。
    *   同一目标短时间内受到连续强击退时，第二次以后击退收益递减，避免被永久浮空或无限撞墙。
    *   超重、泥泞、冰冻、被缠绕等状态会降低可继承动量。
    *   连段失败、撞墙、被格挡或被敌人反击时，玩家会进入短暂硬直，确保高收益动作有明确风险。

## 5. 实战双轨经验获取机制 (Dual-Track Progression)

玩家主属性无法通过常规经验值分配升级，必须依靠白天的地下城战斗，双轨累积武器熟练度（Proficiency）经验与属性本身（Attribute）经验。

### 5.1 实战双轨经验获取大表
在地下城进行探索、战斗时，每一次有效的攻击、闪避或防守动作，都将双轨产出以下经验值：

| 动作类型 | 绑定风格武器 | 熟练度经验 (EXP) 获取规则 | 主属性经验 (EXP) 获取规则 |
| :--- | :--- | :--- | :--- |
| **近战物理攻击** | 单手/双持/持盾/双手近战 | 造成多少点最终物理伤害，则获得等量近战武器熟练度经验。 | 造成多少点最终物理伤害，则获得等量 **【力量 (STR) 属性经验】**。 |
| **远程物理攻击** | 长弓/轻弩 | 造成多少点最终远程伤害，则获得等量远程武器熟练度经验。 | 造成多少点最终远程伤害，则获得等量 **【敏捷 (DEX) 属性经验】**。 |
| **魔法攻击** | 法杖/魔导书 | 造成多少点最终法术伤害，则获得等量魔法熟练度经验。 | 造成多少点最终法术伤害，则获得等量 **【魔力 (MAG) 属性经验】**。 |
| **盾牌防守格挡** | 防盾 | 成功格挡或减免多少点物理伤害，则获得等量持盾熟练度经验。 | 成功格挡或减免多少点物理伤害，则获得等量 **【体质 (CON) 属性经验】**。 |
| **装甲承受伤害 (重甲)** | 重型铠甲 | 承受最终伤害（减免后）时，获得等量重装甲熟练度经验。 | 每次重装甲承受伤害被击中，直接获得等量 **【体质 (CON) 属性经验】**。 |
| **装甲承受伤害 (轻甲)** | 轻型铠甲 | 承受最终伤害（减免后）时，获得等量轻装甲熟练度经验。 | 每次穿着轻甲承受伤害被击中，直接获得等量 **【灵巧 (AGI) 属性经验】**。 |

*   **属性升级双轨**：主属性（STR、DEX、MAG、CON、AGI、PER）通过实战中动作经验累积达到升级门槛时直接 $+1$ 点；同时，当绑定的熟练度升级时，主属性也会获得固定的 $+1$ 反哺提升。

### 5.2 一至三阶装甲熟练度被动阶梯加成 (Proficiency Perks Matrix)
玩家的防具熟练度等级（一阶、二阶、三阶）提升后，将逐层解锁专属的永久被动加成，实现防御流派的战术深度分化：

#### A. 轻型装甲熟练度阶梯被动 (Light Armor Perks)
*   **【一阶轻装被动：轻盈步法 / Light Footwork】**：穿着轻型铠甲时，**基础闪避率额外 $+2\%$**。
*   **【二阶轻装被动：侧身闪避 / Sidestep】**：穿着轻型铠甲时，**基础闪避率额外 $+4\%$，且侧击与背击对玩家造成的命中加成降低 $50\%$**。
*   **【三阶轻装被动：疾风闪跃 / Gale Evasion】**：穿着轻型铠甲时，**基础闪避率额外 $+6\%$，受到的暴击率降低 $5\%$，且成功闪避怪物的物理攻击后，自身下一次动作消耗的行动 Tick 降低 $15\%$**。

#### B. 重型装甲熟练度阶梯被动 (Heavy Armor Perks)
*   **【一阶重装被动：装甲加固 / Reinforced Armor】**：穿着重型铠甲时，**最终物理防御值额外 $+2$**。
*   **【二阶重装被动：冲击缓冲 / Impact Cushioning】**：穿着重型铠甲时，**最终物理防御值额外 $+5$，且受到物理攻击时有 $20\%$ 的概率使装备防具不扣减耐久度**。
*   **【三阶重装被动：不屈壁垒 / Unyielding Bastion】**：穿着重型铠甲时，**最终物理防御值额外 $+10$，完全免疫任何战术击退效果（无法被击退），且每次承受物理伤害有 $15\%$ 的概率对攻击者产生真实反震伤害，反震数值等同于自身物理防御力的 $50\%$**。

## 6. 物理净化与地底复生机制 (NPC Cleanse & Rebirth Mechanism)

在地下城中，所有 NPC 与魔物均遵循地底古老的复生特权：

### 6.1 物理打醒净化规则
*   **怪物永不妥协**：地牢中的怪物与 NPC 共享同一种生存底线法则，即**怪物与异化者永远不会投降，战斗判定不存在任何非致死判定或击昏投降设定**。NPC与普通魔物均会撕咬缠斗直至最后一刻。
*   **直接击杀物理净化**：野外遭遇失序的任务 NPC 或异化的地区领主时，玩家无需保留实力，必须在战斗中将其**直接击杀**。
*   **死后重生净化**：击杀失序 NPC 后，其魂体会消散，并遵循地底复生规则于其所属的野外巢穴、祭坛或领地中物理重塑肉身。在其重塑复生的过程中，导致其狂暴异化的“失序症执念”会被物理彻底打散并净化。
*   **理智回归与合作**：次日傍晚，已恢复理智的 NPC 会自主登门造访酒馆柜台，对玩家在野外的“物理净化”表达由衷的感谢，其神智将百分之百恢复，后续剧情与合作（包括徽记交付）分支将正常开启。
