## 生成规则

- 参考包只用于观察比例、分层、轮廓和道具密度。
- 项目内模型仍使用 `1m = 32px`、`1px = 1/32m` 的自建体素盒。
- 新建或重做模型默认不允许体素正体积重叠；装饰层应贴在主体外侧面接触，避免重叠闪烁。
